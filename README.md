#### General inputs:<br/>
  Pause - "P"<br/>
  Reset - "R"<br/>
<br/>

  Right - "D"<br/>
  Fire - Spacebar<br/>
<br/>
#### Player 2 inputs:<br/>
  Left - "<"<br/>
  Right - ">"<br/>
