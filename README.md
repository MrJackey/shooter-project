# Invaders Of Space
Our small project, creating a space shooter game based on Space Invaders

## Inputs
#### General inputs:<br/>
  Pause - "P"<br/>
  Reset - "R"<br/>
<br/>
#### Player 1 inputs:<br/>
  Left - "A"<br/>
  Right - "D"<br/>
  Fire - Spacebar<br/>
<br/>
#### Player 2 inputs:<br/>
  Left - "<"<br/>
  Right - ">"<br/>
  Fire - "."<br/>

### Clone the repo
